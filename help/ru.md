# Справка — Project Manager

## Обзор
Project Manager — это менеджер для большого количества проектов в одной папке.
Он сканирует корневую папку проектов, показывает всё в таблице и позволяет
запускать Claude Code CLI или Codex CLI в нужном проекте одной кнопкой.

Основные возможности:
• Таблица всех проектов с сортировкой и поиском
• AI-анализ проекта через DeepSeek (что это и в каком состоянии)
• Запуск Claude / Codex в выбранном проекте
• Открытие нескольких проектов вкладками в Windows Terminal (пресеты)
• Закрепление важных проектов наверху
• Трекер задач с напоминаниями
• Мультиязычный интерфейс (5 языков), переключается мгновенно

Подсказка: нажмите F1 в любой момент, чтобы открыть эту справку.

## Список проектов
Вкладка «Проекты» показывает все папки из корня проектов.

• Поиск — поле вверху фильтрует по названию, описанию и стеку.
• Сортировка — клик по заголовку столбца.
• Двойной клик по проекту — запуск Claude Code.
• Правая кнопка мыши — контекстное меню со всеми действиями.

Кнопка «🔄 Скан (forced)» пересканирует папку заново, игнорируя кэш.
Кнопка «📂 Папка» открывает корневую папку проектов в Проводнике.
Кнопка «📁 Данные» открывает папку, где лежат настройки и база (%APPDATA%).

## Закрепление и порядок
Закреплённые проекты всегда показываются вверху списка, выделены жёлтым.

• Закрепить / открепить — кнопка «📌 Закрепить» или контекстное меню.
• Изменить порядок — перетащите закреплённый проект мышкой вверх или вниз,
  либо используйте Alt+↑ / Alt+↓.

Порядок сохраняется автоматически и переживает перезапуск программы.

## Запуск Claude и Codex
Выберите проект и нажмите «▶ Claude» или «▶ Codex» — откроется новое окно
терминала с агентом, уже запущенным в папке проекта.

Запуск использует скрипты с рабочего стола (Claude-BypassProxy, Codex-BypassProxy),
которые поднимают нужное окружение и обходят прокси.

«✨ Новый» создаёт новую папку проекта и сразу запускает в ней агента.

## Вкладки терминала и пресеты
Кнопка «🖥 Открыть во вкладках» открывает диалог запуска нескольких проектов
сразу — каждый в отдельной вкладке Windows Terminal со своим цветом и заголовком.

Как работать:
1. Пометьте нужные проекты (контекстное меню → «Пометить для терминала»).
2. Откройте диалог, при необходимости измените порядок и заголовки вкладок.
3. Нажмите «🚀 Открыть».

Пресет — это сохранённый набор проектов. Сохраните его кнопкой
«💾 Сохранить как…», и в следующий раз откройте весь набор одним кликом.
Это удобно для утреннего «открой всё, над чем я сейчас работаю».

Заголовок вкладки можно задать двойным кликом в списке или в контекстном
меню основной таблицы («Заголовок вкладки терминала…»). Заголовок также
виден в столбце «Заголовок» главной таблицы.

## Восстановление заголовков
Claude после команды /resume переписывает заголовок вкладки терминала.
Кнопка «🏷 Восстановить заголовки» возвращает заголовки на место.

Программа находит открытые вкладки терминала, определяет, какой проект
в каждой из них, и переименовывает вкладки обратно в их настроенные
заголовки. Привязки к пресету не требуется — определение динамическое.

## Трекер задач
Вкладка «Трекер задач» — это список задач, идей и заметок по проектам.

• Задача может быть привязана к проекту или быть «без проекта».
• У задачи есть тип, статус, приоритет, теги, дедлайн и напоминание.
• Напоминание срабатывает внутри программы; можно также создать
  системное напоминание Windows (через Планировщик задач) — оно сработает,
  даже если программа закрыта.
• Кнопки «🚀 Тест в Claude / Codex» создают папку для идеи, пишут IDEA.md
  и запускают агента для проработки идеи.

## DeepSeek-анализ
Анализ описывает, что за проект, как он работает и на какой стадии.

• «🤖 DS-анализ» — анализ выбранного проекта.
• «🤖 DS: новые» — анализ только проектов без описания.
• «🤖 DS: всё» — повторный анализ всех проектов.
• «⏹ Стоп» — прервать массовый анализ.

Результат сохраняется в кэш и показывается в правой панели и в таблице.

## Настройки и язык
• Размер шрифта — кнопки A− / A+ или Ctrl+колесо мыши.
• Язык — выпадающий список вверху. Интерфейс переключается мгновенно,
  без перезапуска. Доступны русский, английский, немецкий, испанский, китайский.

Все настройки и данные хранятся в %APPDATA%\ProjectManager.
Ежедневная резервная копия — в папке Documents\ProjectManager-Backups.
